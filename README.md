# FitTrack

AI-powered fitness, nutrition, workout tracking and social fitness platform.

## Features

### MVP
- Authentication
- User Profiles
- Workout Tracking
- Nutrition Tracking
- Weight Tracking
- Dashboard Analytics
- Community Feed
- Follow System
- Kudos & Comments

## Tech Stack

### Frontend
- Next.js
- TypeScript
- TailwindCSS
- ShadCN UI

### Backend
- FastAPI
- PostgreSQL
- SQLAlchemy

### Infrastructure
- Docker
- Docker Compose
